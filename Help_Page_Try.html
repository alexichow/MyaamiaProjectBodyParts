Try Program Page Information

The try program allows the user to learn the Myaamia language through exploration.

The try program is an interactive program allowing the user to click and listen to the spoken word for the part clicked on.

To operate the program click on one of the torso parts and listen to the spoken word.

The Ni- Ki- A- person marking cube buttons can be clicked to change the current person marking being used. The current person marking in use will be highlighted on the cube.

Button Information

There is a single cube of buttons used for this program, the Ni- Ki- A- person marking cube

Ni- Ki- A- Cube

The Ni- Ki- A- cube is used to change the person marking being used.

  • Click Ni- to change the audio to use that person marking.
  • Click Ki- to change the audio to use that person marking.
  • Click A- to change the audio to use that person marking.
