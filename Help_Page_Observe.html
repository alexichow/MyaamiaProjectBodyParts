Observe Page Program Information

The observe page allows the user to learn through observation.

When the program is started the parts of the torso will highlight in unison with the audibly spoken word which corresponds to the part highlighted.

All of the parts will be highlighted and played first using the person marking ni-.
Then all parts will be highlighted and played using the person marking ki- and after that
using the person marking a-.

Button Information

There are two button cubes for this program, the ni- ki- a- cube and the audio control cube.

Ni- Ki- A- Cube

The ni- ki- a- cube highlights as the person marking changes.

Audio Control Cube

The audio control cube is used to pause, play, or restart the program.

  • Click play to start the program if it did not start automatically.
  • Click pause to pause the program then click play to unpause it.
  • Click restart button to completely restart the program.
