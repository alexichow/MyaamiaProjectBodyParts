Test Program Page Information

The test program allows the user to test their understanding of Myaamia language.

If the program does not automatically start press the play button to begin the testing.
To operate the program listen to the spoken word and then click the part you think
corresponds to that word.

With each correct match a green check mark image will show and a ding sound will play.
With each incorrect match a red x image will show and a rattle sound will play.

At the end of the testing you will be given a total score and asked if you would like to
test again or stop testing.

  • If you choose to continue the testing will start over again.
  • If you choose to stop, you will be automatically redirected to the main 123 page.

Button Information

There is a single cube of buttons used for this program, the audio control cube.

Audio Control Cube

The audio control cube is used to pause, play, or restart the program.

  • Click play to start the program if it did not start automatically.
  • Click pause to pause the program then click play to unpause it.
    ( Be sure to finish answering the test match to continue.)
  • Click restart button to completely restart the testing program.
